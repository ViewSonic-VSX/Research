# whisper code

Python version
3.11.7

## Installation
```
pip install -r requirements.txt
```

## Command
```
# Show all the microphone Index
main.py -l
```

```
# Fire main.py with selected device index
main.py -d {index}

# For example
main.py -d 1
```
