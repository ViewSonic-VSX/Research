# ControlNet code

Python version
3.11.5

## Installation
```
pip install -r requirements.txt
```

## Setup
Setup arguments in main.py

```
device: ["cpu", "cuda", "mps"]
image_path: Uploaded sample image in ./sample
prompt: The prompt that used to generate image
h, w: image height, image width
```

# Run
```
python main.py
```
